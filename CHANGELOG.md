# Changelog

## 11.0.0

- Added full status legends on history, leaderboard, administration and submission pages.
- Harmonized block content typography.
- Updated the TSP C++ sample to print iterative tour-length progress and a final metric line.

## 10.0.0

- Introduces one active project selected from `projects/<project_id>`.
- Adds declarative project profiles with data files, instances, metrics and ranking policy.
- Adds declarative language profiles for C++, Python and Java.
- Adds Docker runner images for C++, Python and Java.
- Generalizes build commands, run commands, stdout parsing, leaderboard columns, CSV exports and PDF reports.
- Adds example project `tsp` for C++ and example project `mnist_digits` for Python digit classification.
- Adds release-oriented files: `LICENSE`, `CHANGELOG.md`, `SECURITY.md`, `CONTRIBUTING.md`, `.gitignore`, JSON schemas and a packaging script.
